# Airlink

Airlink is a web chat application (frontend + Node/Express backend) that provides authenticated messaging and media support.

## Tech

- Frontend: React + Vite, HeroUI, Clerk authentication, Zustand state
- Backend: Node.js + Express, Clerk auth, Socket.IO, database models for users/messages

## Local development

### Backend

1. `cd backend`
2. `npm install`
3. `npm run dev`

### Frontend

1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Environment variables

Check the backend setup for required configuration (e.g., `PORT`, `FRONTEND_URL`, and Clerk-related values).

## Notes

- The UI text/app name is branded as **Airlink** across the frontend.
